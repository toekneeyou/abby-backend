## Getting Started

1. Clone repository

   ```bash
   git clone https://github.com/toekneeyou/abby-backend.git

   ```

2. Install dependencies

   ```bash
   npm i

   ```

3. Create a `.env` file with these variables

   ```
   PORT

   DB_NAME
   DB_HOST
   DB_PORT
   DB_USERNAME
   DB_PASSWORD
   ```
